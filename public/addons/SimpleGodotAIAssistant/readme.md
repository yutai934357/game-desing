# Simple Godot AI Assistant GDScript

This plugin has been rewritten using GDScript.

**Note:** This plugin does not provide code completion and is not aware of what you are currently typing; it only provides a chat interface. Please be aware that the Agent can execute arbitrary code, so use it with caution.

## Recommended Models

For best results, please use this plugin with **Gemini 3 Pro Preview** or **Gemini 3 Flash Preview** models.

**Feature requests are not accepted.** If you require complex features, please look for other plugins.

# Prompt Examples

*   "Create a crystal shader and apply it to the selected node's material while keeping the texture unchanged. Save the file to the `res://Material/` directory."
*   "Create a black hole, place it at the position of the currently selected node, and save it to the `res://VFX/BlackHole/` directory."
*   "Create a ground effect made of magic bricks at the position of the selected node. Initially, there should be nothing; when running, the magic bricks should fly in from all sides to form the path. The effect should last for 10 seconds. Save the relevant scripts to the `res://VFX/` directory."


# Simple Godot AI Assistant GDScript

该插件已使用GDScript重写。

这个插件不能帮你补全代码，也不了解你正在输入的内容，它只提供一个聊天框。请注意，Agent可以执行任意代码，因此使用时必须小心。

## 推荐模型

为了达到最佳效果，请搭配Gemini 3 Pro preview或Gemini 3 Flash Preview模型使用。

不接受功能请求。如果你需要复杂的功能，请寻找其它插件。

# 提示词示例

创建一个水晶着色器，修改选中节点的材质，保持纹理不变。将文件保存到res://Material/目录。

创建一个黑洞，放置到当前选中节点的位置，保存到res://VFX/BlackHole/目录。

在选中节点的位置创建一个由魔法砖块铺成的地面特效，初始时空无一物，运行时魔法砖块从四周飞来铺成路面。特效持续10秒。相关脚本保存到res://VFX/目录。
